Free Return Lunar Mission

Fall 2023

Overview

• Goal of this project was to design a free return lunar trajectory mission for a spacecraft
• Overall mission duration and closest approach to moon requirements had to be met
• Spacecraft was allowed one burn to direct it to the moon — this Δv had to be determined

Methods

• Simulated various mission trajectories using NASA's General Mission Analysis Tool (GMAT) and its differential solver
• Performed iterative procedure to propogate potential trajectories until suitable path and Δv was found
• Utilized atmospheric drag paramters to simulate spacecraft's reenetry into Earth's atmosphere

Results

• Obtained required Δv and orbital details of when burn should occur, given the real positions of Earth and Moon during theoretical spacecraft launch date
• Obtained free return trajectory flight path of spacecraft
• Mission duration was 2 days within given time frame

Scooter Design

Spring 2023

Overview

• Goal of this project was to design an original kick scooter with front wheel suspension
• Required strength and allowable deflections of platform and handlebar were specfied, along with budget constraint

• Modeled and assembled components of a kick scooter with suspension using SolidWorks
• Performed FEM analysis to ensure design met given stress/deflection constraints.

• Obtained some pre designed components using the McMaster-Carr manufacturaing site
